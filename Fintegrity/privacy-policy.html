About MoneyCanva

MoneyCanva is a privacy-first financial management application that operates entirely offline. Users can:

  • Track expenses and income locally on their device
  • Manage multiple financial accounts (3 in free tier)
  • Generate financial reports without cloud processing
  • Upgrade to premium for additional features (no personal data required)

Information Collection

MoneyCanva does not collect or process any personal information:

  • No registration required for basic features
  • All financial data stays exclusively on your device
  • No usage analytics or tracking mechanisms
  • No connection to external servers or cloud services

Premium Features

The optional premium upgrade via in-app purchase:

  • Does not require sharing personal information
  • Uses platform-native payment systems (Apple/Google)
  • Does not create user accounts or profiles

Data Security

Your financial data is protected through:

  • Device-native encryption systems
  • No internet permission requirements
  • Optional local backup (user-controlled)

Third-Party Services

MoneyCanva contains no third-party:

  • Analytics services
  • Tracking libraries
